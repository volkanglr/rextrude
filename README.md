# Rextrude

Turns photos into a 3D model with AI

>early developement

## What it does

Rextrude takes one photo (or more photos, depending on the used AI model), sends it to an AI 3D-generation service and returns a usable 3D mesh (.glb). Previewable in the browser and downloadable for usage in external programs. (e.g. Blender, Cinema4d,..)
